# Astro 入门指南

Astro 是一个现代化的静态站点生成器，它让我重新思考了如何构建 Web 应用。

## 为什么选择 Astro？

1. **零 JavaScript 默认** - 页面加载速度极快
2. **群岛架构** - 只在需要的地方使用 JavaScript
3. **框架无关** - 支持 React、Vue、Svelte 等

## 快速开始

```bash
npm create astro@latest
```

## 我的使用体验

使用 Astro 构建这个个人站点后，我深刻体会到了它的优势：

- 构建速度非常快
- 生成的页面体积很小
- Markdown 支持开箱即用

## 结语

如果你正在寻找一个简单高效的静态站点方案，Astro 值得一试。
